A startup company called FitTrack is developing a fitness application to help users monitor their daily health and 
fitness activities. Each user has their own fitness record, which includes their name, the number of steps walked, 
calories burned, and hours slept in a day.

Design a Java program using Classes and Objects to model the fitness tracking system. 
The program should store and display the fitness details of a user, determine whether the daily step goal of 10,000 steps 
has been achieved, and check whether the user has a healthy sleep duration of at least 7 hours.

The program should support multiple users by creating separate objects for each user.

Requirements

The program should:

Store the user's name, steps walked, calories burned, and hours slept.
Display the complete fitness summary.
Check whether the user has achieved the daily step goal (10,000 steps).
Check whether the user has healthy sleep habits (at least 7 hours of sleep).
Support multiple users independently using objects.

Sample Input 1
Enter User Name:
Rahul

Enter Steps Walked:
12500

Enter Calories Burned:
650

Enter Hours Slept:
8

Sample Output 1
----- Fitness Summary -----
User Name        : Rahul
Steps Walked     : 12500
Calories Burned  : 650
Hours Slept      : 8

Step Goal Achieved
Healthy Sleep Duration

Sample Input 2
Enter User Name:
Priya

Enter Steps Walked:
7200

Enter Calories Burned:
430

Enter Hours Slept:
5
Sample Output 2
----- Fitness Summary -----
User Name        : Priya
Steps Walked     : 7200
Calories Burned  : 430
Hours Slept      : 5

Step Goal Not Achieved
Sleep Duration Below Recommended Level
